# Quartz Website Design — Creative Minimalist

## Design Goals

- **Minimalist:** Reduce visual noise. Remove the heavy "Mission Control" block.
- **Depth:** Use "Pro Glass" aesthetic (blur, transparency, shadows) for panels.
- **Creative:** Switch Inspector to a floating side panel (like Settings).
- **Pro:** High-density information with clearer hierarchy.

## Rules

- Do NOT change background colors with Tailwind — only use the bgn colors from `app.css`
- Use Tailwind for everything else (spacing, typography, layout)
- Get down to the nitty gritty details — be specific, not surface level
- Don't make changes to the codebase — only the design spec
